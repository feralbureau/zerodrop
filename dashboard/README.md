# shadcn/ui Vite app

This is a standalone Vite app with shadcn/ui.

## Adding components

To add components, run the following command at the project root:

```bash
npx shadcn@latest add button
```

This will place the ui components in the `src/components` directory.

## Using components

To use the components in your app, import them from the ui alias.

```tsx
import { Button } from "@workspace/ui/components/button";
```
